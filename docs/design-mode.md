# Design Mode

Design Mode lets you create a complete design system through an interactive conversation with a Designer Agent.

---

## Workflow

### 1. Category Selection

Choose from 8 design categories:

- Blog
- Landing Page
- E-commerce
- Dashboard
- Documentation
- Portfolio
- Chat / Messaging
- SaaS Marketing

### 2. Discovery Phase

Chat with the Designer Agent about your project's visual needs, brand identity, target audience, and design preferences.

### 3. Theme Generation

The agent generates multiple theme options (color palettes, typography). You can:

- **Select** a theme
- **Refine** a theme with feedback
- **Request new options**

### 4. Component Generation

Based on the selected theme, the agent generates component styles (cards, buttons, inputs, badges, etc.). Review and refine as needed.

### 5. Mockup Generation

Full-page mockups are generated as visual HTML previews. You can:

- Select from multiple options
- Enter **Refine Mode** for side-by-side editing
- Request entirely new options

### 6. Pages

Create multiple pages for your design system. Add pages one at a time, each generated based on your theme and components.

### 7. Save to Library

Name your design and save it to your design library. Saved designs can be attached to [projects](projects.md) in any workspace to guide implementation.

## Design Library

Access your saved designs from the **Designs Library** page. Each design includes:

- Color tokens and palette
- Typography definitions
- Spacing scale
- Component styles
- Page mockups (as HTML)

## Attaching Designs to Projects

In [project settings](projects.md), you can attach a design from your library. When the agent implements UI features for that project, it will reference the design tokens and guidelines.
