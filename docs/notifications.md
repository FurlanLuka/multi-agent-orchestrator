# Notifications

Orchy supports desktop and audio notifications to keep you informed.

---

## Settings

Click the **notification bell** icon in the session header to configure:

| Setting | Description |
|---------|-------------|
| **Desktop Notifications** | Browser push notifications for status changes |
| **Audio Notifications** | Sound alerts for important events |

Notifications are useful when Orchy is running in the background — you'll be alerted when the agent needs your input or when tasks complete.
